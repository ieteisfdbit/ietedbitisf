Iete-Dbit-Isf Events


Events


2016

The IETE-DBIT-ISF conducted Industrial Visit to Indian Institute of Geomagnetisim, Panvel. It alsos conducted workshop on FUN WITH FM to develop a FM Transmitter using simple components such as transistors and LC circuit; workshop on Arduino to make students learn a new open-source project that can create microcontroller based kits for building digital devices and interactive objects that can sense and control physical devices; workshop and competition on Line Follower to make students learn how to use a new open-source project for creating a Arduino based Line Follower Robot; workshop on Python. It also conducted a workshop on "Raspberry Pi: Basics, GPIO Interfacing and IOT" to program the GPIO pins of R-Pi for device status & control, to perform various Digital & Analog controls via R-Pi and to use the R-Pi on the internet as a web server.

2017

A workshop on Node MCU ESP8266 IoT WiFi Module To learn about NodeMCU ESP 8266, to read sensor data & control a device on Intranet(LAN), to read sensor data & control a device on Internet using Thingspeak & ThingHTTP and to incorporate MQTT protocol and IFTTT for IoT application was conducted. The Valedictory programme of IETE Mumbai Oscillations 2017 was conducted at Don Bosco Institute of Technology, Mumbai to felicitate the winners of various events held at the ISF’s of IETE Mumbai spread across different engineering colleges in Mumbai under the banner of Oscillations 2017. Arduino Workshop to learn about Arduino Nano Module and to program the controller to interface the module with various peripherals and sensors was held. Circuit Simulation and PCB designing Workshop to learn how to simulate analog and digital circuits and to design the layout for the circuit using software was held. A workshop on Node MIT APP INVENTOR to get acquainted with app development using the MIT App inventor tool and to develop an application for home automation using Bluetooth communication was conducted.

2018

A WiFi Controlled Bot workshop and competition was held. It helped to learn about NodeMCU ESP 8266 and MIT APP inventor and to make a robot interfaced with NODEMCU-ESP8266 while controlling the robot using an ANDROID application. Workshop on Node-Red software helped the students to understand NodeRed, JSON Messages and to get acquainted with the GUI using data from MQTT & Serial Communication. An Intercollegiate Circuit Designing Competition was conducted. A workshop on Internet of Things using Node MCU ESP8266 WiFi Module was held. Arduino: Basics, Interfacing & Control using Python GUI to learn about Arduino Nano Module and to program the controller to interface the module with various peripherals and sensors.

Read more


2019

The Robowars Chapter#1 Sumo Wrestle Workshop: An IOT-ROBOT building workshop and competition was held. The aim was to learn about Arduino NANO Microcontroller and the Bluetooth HC-05 module, to make a robot interfaced with Arduino NANO module and to control the robot using a Bluetooth application. Circuit Simulation and PCB Designing workshop and competition was also conducted. It helped the students to learn how to simulate circuits using Proteus and to use Eagle software to create schematic of circuits and convert them to a layout suitable for PCB using manual and auto routing while getting the layout of the circuit onto a PCB, etching, drilling, soldering.

2020

The Drone Making event was conducted where students had to assemble the components, install the app for flying drone on eduvance website and then connect the phone to the drone, in order to fly the drones. ROBOSOCCER WORKSHOP to learn about ESP 8266 Microcontroller, learn to make a robot interfaced with NODE MCU module and to control the robot using a WIFI application was held. Career options & oppurtunities for B.E., EDT and EC graduates webinar, consulting career path with an engineering background webinar, emerging Trends Post COVID-19 webinar, webinar on Physics and Engineering, Webinar on “Critical Media Literacy”, Webinar on 'Open Source Technology: A Step Towards Collaborative Learning' was held.

2021

Bug Busters Competition, Poster Making Competition, Technical Writing Competition and an online workshop on “Introduction to Microbit using Tinkercad" was held. Introduction to Microbit using the online simulator Tinkercad was to learn about the basics of controllers and processors , various sensors , circuit designing skills and learn some hands-on application to use in real time.
